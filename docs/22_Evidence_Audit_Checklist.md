# SDKA Evidence Audit Checklist
**File:** docs/22_Evidence_Audit_Checklist.md

Version: 1.0.0  
Status: Active Standard

---

# Purpose

This checklist is the mandatory quality gate for every SDKA Knowledge Object before it is marked **Frozen** (Repository Ready).

A Knowledge Object may progress through research stages, but it cannot reach **R3 – Repository Ready** until this checklist has been completed.

---

# 1. Topic Definition

- [ ] Knowledge Object ID assigned
- [ ] Title finalized
- [ ] Scope defined
- [ ] Research questions documented
- [ ] Out-of-scope items recorded

---

# 2. Terminology Audit

- [ ] Primary terminology verified
- [ ] Alternate names documented
- [ ] Regional terminology recorded
- [ ] Sanskrit spellings checked
- [ ] Ambiguous terms explained

---

# 3. Primary Source Audit

- [ ] Primary texts identified
- [ ] Exact references recorded where available
- [ ] Quotations verified against the edition used
- [ ] Unsupported claims removed
- [ ] Missing primary evidence documented

---

# 4. Secondary Source Audit

- [ ] Scholarly references compared
- [ ] Conflicting interpretations recorded
- [ ] Academic consensus summarized
- [ ] Source quality assessed

---

# 5. Living Tradition Audit

- [ ] Temple practice documented
- [ ] Regional customs documented
- [ ] Oral traditions identified
- [ ] Guru-parampara boundaries respected

---

# 6. Historical Audit

- [ ] Historical development described
- [ ] Archaeological evidence reviewed
- [ ] Inscriptions referenced where available
- [ ] Timeline prepared

---

# 7. Evidence Matrix Audit

Every major statement should have:

- [ ] Evidence level (A–D)
- [ ] Confidence rating
- [ ] Source classification
- [ ] Verification status

---

# 8. Misconception Audit

- [ ] Common myths identified
- [ ] Unsupported claims rejected
- [ ] Regional differences clarified
- [ ] Internet claims validated before inclusion

---

# 9. Repository Audit

Required package files:

- [ ] README.md
- [ ] metadata.yaml
- [ ] primary-sources.md
- [ ] architecture.md (if applicable)
- [ ] ritual.md (if applicable)
- [ ] historical-development.md
- [ ] symbolism.md
- [ ] living-traditions.md
- [ ] misconceptions.md
- [ ] evidence-matrix.md
- [ ] bibliography.md
- [ ] knowledge-graph.md
- [ ] CHANGELOG.md
- [ ] content/instagram.md

---

# 10. Knowledge Graph Audit

- [ ] Related Knowledge Objects linked
- [ ] New nodes identified
- [ ] Relationships documented
- [ ] Future links proposed

---

# 11. Content Readiness

- [ ] Instagram content derived
- [ ] Evidence mapped to content
- [ ] Image prompts prepared
- [ ] Caption reviewed
- [ ] CTA prepared

---

# 12. SDKA Quality Gate

A Knowledge Object is **Frozen** only when:

- [ ] At least one verified primary source exists (or absence is documented)
- [ ] Secondary literature reviewed
- [ ] Evidence matrix completed
- [ ] Bibliography completed
- [ ] Regional variation documented
- [ ] Misconceptions documented
- [ ] Open research questions recorded
- [ ] Knowledge graph updated
- [ ] Repository package complete
