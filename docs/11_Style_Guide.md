# Purpose

This guide defines SDKA's voice, language standards, typography, color palette, and naming conventions. All publications—carousels, articles, scripts—must follow this guide.

---

# Voice & Tone

## Core Voice Attributes

| Attribute | Description | Example |
|-----------|-------------|---------|
| Scholarly | Evidence-based, precise | "According to the Valmiki Ramayana (Sundara Kanda 1.1)..." |
| Warm | Inviting, not academic-cold | "Have you ever wondered why..." |
| Respectful | Honors all traditions | "In the Sri Vaishnava tradition..." |
| Curious | Questions drive content | "What did the earliest sources actually say?" |
| Humble | Acknowledges limits | "The historical evidence here is limited." |

## What SDKA Sounds Like

- A knowledgeable friend who has done deep research
- A museum guide who makes ancient wisdom accessible
- A researcher who respects tradition and evidence equally

## What SDKA Does NOT Sound Like

- A mythology fan page ("You won't BELIEVE what Shiva did!")
- A debate club ("This tradition is WRONG about...")
- A textbook ("The aforementioned deity manifestation...")
- A motivational speaker ("Unlock the SECRET power of...")

# Language Rules

## General

- Use clear, modern English. Avoid unnecessary Sanskrit in body text.
- Introduce Sanskrit terms with transliteration and brief definition on first use.
- Prefer active voice.
- Keep carousel slide text under 40 words per slide.
- Keep captions under 2,200 characters (Instagram limit).

## Sanskrit & Transliteration

- Use **IAST transliteration** (ā, ī, ū, ṛ, ṣ, ṅ, ñ, etc.) in formal content.
- Simplified transliteration (aa, ii) acceptable in casual captions if consistent.
- Deity names: use most widely recognized form (Shiva, Vishnu, Devi, Hanuman).
- Include Devanagari on title slides or when the script itself is the subject.

## Evidence in Prose

When stating claims:

- **Strong claim:** "The Valmiki Ramayana describes Hanuman as..."
- **Traditional claim:** "Temple tradition in Karnataka holds that..."
- **Uncertain:** "The origins of this practice remain debated among scholars."
- **Folklore:** "Popular tradition tells that... (this is not found in early scriptures)."

## Prohibited Language

- Sensationalism: "shocking," "secret," "they don't want you to know"
- Disrespect: "myth," "superstition," "primitive" (when referring to living traditions)
- False certainty: "definitely," "proven fact" (without evidence tag)
- Clickbait: "You won't believe," "This changes everything"

# Naming Conventions

## Deities

Use the most recognizable English form. Note regional variants in KO, not in every caption.

| Standard | Variants (use in regional context) |
|----------|--------------------------------------|
| Shiva | Shankara, Mahadeva, Rudra |
| Vishnu | Narayana, Hari |
| Devi | Shakti, Amba |
| Hanuman | Anjaneya, Maruti, Bajrangbali |
| Ganesha | Ganapati, Vinayaka |
| Murugan | Kartikeya, Skanda, Subrahmanya |

## Texts

| Standard | Notes |
|----------|-------|
| Valmiki Ramayana | Not "Ramayan" in formal content |
| Mahabharata | |
| Bhagavad Gita | Not "Gita" alone on first reference |
| Vedas | Rigveda, Yajurveda, Samaveda, Atharvaveda |

## Places

Use current official names with traditional name in parentheses on first reference if different.

"Varanasi (Kashi)" — then either form is acceptable.

# Typography

## Primary Fonts (Recommended)

| Use | Font | Weight |
|-----|------|--------|
| Headlines | Playfair Display / Cormorant Garamond | Bold |
| Body text | Inter / Source Sans Pro | Regular |
| Sanskrit | Noto Sans Devanagari | Regular |
| Captions | Inter | Regular, 14–16px equivalent |

## Sizing (Instagram 1080×1350)

| Element | Size |
|---------|------|
| Slide headline | 48–64 pt |
| Slide body | 28–36 pt |
| Sanskrit text | 32–40 pt |
| Caption text | 14–16 pt (in-app) |

## Rules

- Maximum 2 fonts per carousel.
- Headlines: sentence case, not ALL CAPS (except single emphasis words).
- Body text: left-aligned or center-aligned per archetype; never justified on narrow slides.
- Sufficient contrast: text must pass WCAG AA on its background.

# Color Palette

## Primary Palette

| Name | Hex | Use |
|------|-----|-----|
| Saffron | `#FF6F00` | Accents, highlights, CTAs |
| Deep Indigo | `#1A237E` | Backgrounds, headers |
| Warm Gold | `#FFC107` | Borders, decorative elements |
| Parchment | `#FFF8E1` | Light backgrounds, text slides |
| Charcoal | `#212121` | Body text on light backgrounds |

## Secondary Palette

| Name | Hex | Use |
|------|-----|-----|
| Temple Red | `#B71C1C` | Festival, Shakta themes |
| Forest Green | `#1B5E20` | Nature, certain deity themes |
| Sandstone | `#D7CCC8` | Temple, architecture themes |
| Midnight | `#0D1117` | Dark mode slides |

## Usage Rules

- One dominant color per carousel (from primary palette).
- Secondary colors for thematic variation, not rainbow mixing.
- Saffron + Deep Indigo is the default SDKA combination.
- Maintain sufficient contrast for readability.

# Visual Identity Elements

## Logo & Mark

- SDKA wordmark: clean serif, saffron on dark or indigo on light.
- Optional: Om or lamp motif as subtle watermark—not dominant.

## Slide Layout

- Consistent margins: 80px minimum on all sides (at 1080 width).
- Bottom bar: series name or episode number (small, consistent position).
- Source attribution: bottom-left on Revelation and Deep Dive slides when applicable.

## Photography & Illustration

- Prefer warm, natural lighting for temple photography.
- Illustrations: traditional art styles per `06_Visual_System.md`.
- No stock photos of generic "Indian spirituality."
- No AI-generated deity images for publication without art direction approval.

---

# Hashtag Guidelines

- 5–10 hashtags per Instagram post.
- Mix: 2 broad (#SanatanaDharma, #Hinduism), 3 topic-specific, 2 niche.
- No spam hashtags (#love #instagood).
- Create branded tag: `#SDKA` or project-specific tag.

# Caption Structure

```
{Hook — one compelling sentence or question}

{2–3 sentences of context and key insight}

{Evidence note if strong claim: "According to [source]..."}

{Connection: "This connects to [related topic] — more in our [series name] series."}

{CTA: "Save this for later" or "Follow for more on [theme]."}

{Hashtags}
```

---

# Accessibility

- Alt text on every Instagram image: describe the visual content, not just the topic.
- Sufficient color contrast on all text slides.
- Avoid conveying meaning through color alone.
- Sanskrit terms: include transliteration in alt text.
