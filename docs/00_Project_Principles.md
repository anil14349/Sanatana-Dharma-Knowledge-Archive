# 00 – Project Principles (Quick Reference)

> **Research First → Understanding Second → Storytelling Third → Virality Last**

## 5 Non-Negotiable Rules

1. **Cite before you claim.** Primary scripture → commentarial tradition → academic scholarship → popular sources. No unsourced claims.
2. **Document, don't judge.** Present what a tradition says before comparing. The archive educates; it does not persuade.
3. **Say "we don't know."** When evidence is uncertain or traditions disagree, state that openly.
4. **No sensationalism.** Never distort teachings for engagement. Never ridicule beliefs. No political positioning.
5. **IAST always.** All Sanskrit uses proper diacritics (ā, ī, ū, ṛ, ṣ, ś, ṇ, ṭ, ḍ, ñ). No exceptions.

## Decision Test

Before publishing any content, ask:

- Does this serve **understanding** or just **engagement**?
- Is every claim **backed by a cited source**?
- Would a scholar find this **respectful and accurate**?

If any answer is no, revise.
