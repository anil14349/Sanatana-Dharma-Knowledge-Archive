# Verified Primary Source

## Kamika Agama

**Status:** Partially Verified

Current research supports that the Kamika Agama discusses temple circulation, ritual architecture, and the significance of the Soma Sutra within Shaiva temple worship.

### Current Repository Position

- Supports ritual circulation in Shaiva temples.
- Supports architectural significance of the Soma Sutra (Pranala).
- Supports Agamic relationship between temple layout and ritual.
